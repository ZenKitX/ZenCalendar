# ZenCalendar 项目总结

## 📱 项目概述

ZenCalendar 是一个简洁优雅的日历和意图管理应用，融合了禅意设计理念和现代 Material 3 设计语言。应用旨在帮助用户更好地管理时间、设定每日意图，并通过禅语获得启发。

## 🎯 核心功能

### 1. 日历管理
- 月视图日历（table_calendar）
- 事件创建、编辑、删除
- 事件详情查看
- 全天事件支持
- 事件时间管理
- 日期快速导航

### 2. 意图管理
- 每日意图设定
- 意图完成追踪
- 历史记录查看
- 完成统计和进度
- 滑动删除功能

### 3. 禅语系统
- 每日禅语显示
- 多种禅语分类
- 随机刷新功能
- 基于日期的固定禅语

### 4. 设置管理
- 主题切换（浅色/深色/跟随系统）
- 通知设置
- 数据统计
- 数据清除
- 关于信息

## 🏗️ 技术架构

### 架构模式
**GetX MVC + Clean Architecture 精简版**

```
lib/
├── main.dart                           # 应用入口
└── app/
    ├── config/                         # 配置
    │   └── theme/                      # 主题配置
    │       ├── zen_colors.dart         # 颜色定义
    │       └── app_theme.dart          # 主题配置
    ├── core/                           # 核心
    │   └── init_dependencies.dart      # 依赖注入初始化
    ├── data/                           # 数据层
    │   ├── models/                     # 数据模型
    │   │   ├── event_model.dart
    │   │   ├── intention_model.dart
    │   │   └── zen_quote_model.dart
    │   ├── providers/                  # 数据提供者
    │   │   └── local_storage_provider.dart
    │   └── repositories/               # 仓库
    │       ├── event_repository.dart
    │       ├── intention_repository.dart
    │       └── quote_repository.dart
    ├── modules/                        # 功能模块
    │   ├── home/                       # 主页（底部导航）
    │   ├── calendar/                   # 日历模块
    │   ├── intention/                  # 意图模块
    │   └── settings/                   # 设置模块
    ├── routes/                         # 路由
    │   ├── app_routes.dart
    │   └── app_pages.dart
    └── services/                       # 服务
        └── haptic_service.dart         # 触觉反馈服务
```

## 🎨 设计系统

### 设计理念
**Soft UI Evolution（柔和 UI 进化）**

继承自 Calculator 项目的 Neumorphic/Soft UI 设计，结合禅意美学：
- 柔和的阴影效果
- 流畅的动画过渡
- 触觉反馈系统
- 禅语启发系统

### 配色方案

#### 浅色主题
- **主色**: 薰衣草紫 (#9C88D4)
- **次要色**: 柔和粉 (#E8B4D4)
- **强调色**: 健康绿 (#7CB342)
- **背景**: 温暖白 (#F5F5F5)

#### 深色主题
- **主色**: 竹林绿 (#4A7C59)
- **次要色**: 月光蓝 (#5C7C8C)
- **强调色**: 禅意金 (#D4AF37)
- **背景**: 深邃黑 (#121212)

### 字体系统
- **标题**: Lora (衬线字体，优雅)
- **正文**: Raleway (无衬线字体，现代)

### 设计元素
- **圆角**: 12-16px
- **阴影**: 柔和多层阴影
- **间距**: 8px 基础单位
- **动画**: 200-300ms 过渡
