## 🎉 重大更新

### 冥想计时器
全新的禅意功能，帮助你保持身心健康：

**计时功能**:
- 预设时长：5/10/15/20/30 分钟
- 开始/暂停/停止控制
- 圆形进度指示器
- 实时倒计时显示

**记录统计**:
- 总冥想时长追踪
- 冥想次数统计
- 连续打卡天数
- 历史记录查看
- 删除单条记录

**音效系统**:
- 开始铃声提示
- 结束铃声提醒
- 系统音效集成

### 高级设置
强大的个性化设置，让应用更符合你的使用习惯：

**语言设置**:
- 中文/English 切换
- 实时语言更新
- 无需重启应用

**显示设置**:
- 字体大小调节（80% - 150%）
- 实时预览效果
- 7档精细调节

**动画设置**:
- 动画速度控制（0.5x - 2.0x）
- 5档速度选择（很慢/慢/正常/快/很快）
- 适应不同设备性能

**主题自定义**:
- 6种预设颜色
  - 蓝色（默认）
  - 紫色
  - 绿色
  - 橙色
  - 粉色
  - 青色
- 颜色选择器
- 重启后生效

**首页设置**:
- 日历视图作为首页
- 意图视图作为首页
- 自由选择默认页面

**数据同步**:
- 自动备份开关
- 每天自动备份数据
- 云端同步（未来版本）

**设置管理**:
- 重置所有设置
- 恢复默认值
- 一键重置
