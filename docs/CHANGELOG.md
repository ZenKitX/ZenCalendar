# 更新日志

## v1.0.0 - 2026-02-28

### 🎉 首个正式版本发布

ZenCalendar 是一个功能完整、设计优雅的禅意日历和意图管理应用。

#### ✨ 核心功能

**日历模块 (Calendar)**
- 📅 月历视图 - 基于 table_calendar 的优雅日历界面
- ➕ 事件创建 - 添加标题、描述、时间的日历事件
- ✏️ 事件编辑 - 修改已有事件的所有信息
- 🗑️ 事件删除 - 带确认对话框的安全删除
- 📋 事件详情 - 完整展示事件信息
- 🔔 触觉反馈 - 操作时的震动反馈增强体验

**意图模块 (Intention)**
- 🎯 每日意图 - 为每天设定专注目标
- ✅ 完成标记 - 标记意图完成状态
- 📊 完成统计 - 显示已完成/总数统计
- 🧘 禅语启发 - 15 条精选禅语随机展示
- 🔄 刷新禅语 - 随时获取新的启发

**设置模块 (Settings)**
- 🌓 主题切换 - 浅色/深色/跟随系统三种模式
- 🔔 通知设置 - 事件提醒和意图提醒开关
- 📊 数据统计 - 显示事件和意图总数
- 🗑️ 数据管理 - 清除所有数据（带确认）
- ℹ️ 关于信息 - 应用介绍和开源许可证

#### 🎨 设计系统

**Soft UI Evolution 风格**
- 配色方案：薰衣草紫 (#8B5CF6) + 健康绿 (#10B981)
- 浅色背景：#FAF5FF（薰衣草白）
- 深色背景：#1F1B24（深紫灰）
- 柔和阴影和现代过渡动画
- 卡片式布局和圆角设计

**字体系统**
- 标题字体：Lora（优雅衬线）
- 正文字体：Raleway（现代无衬线）
- Google Fonts 支持

#### 🏗️ 技术架构

**架构模式**
- GetX 状态管理（MVC 模式）
- Repository 仓库模式
- 依赖注入（Binding）
- 模块化路由系统

**数据持久化**
- SharedPreferences 本地存储
- JSON 序列化/反序列化
- 类型安全的数据模型

**核心依赖**
- Flutter 3.11+
- GetX 4.6.6 - 状态管理和路由
- table_calendar 3.1.2 - 日历组件
- shared_preferences 2.3.3 - 本地存储
- google_fonts 6.2.1 - 字体支持
- intl 0.19.0 - 国际化和日期格式化

#### 📁 项目结构

```
lib/app/
├── config/          # 配置（主题、路由）
├── data/            # 数据层（模型、仓库、存储）
├── modules/         # 功能模块
│   ├── calendar/    # 日历模块
│   ├── intention/   # 意图模块
│   └── settings/    # 设置模块
├── routes/          # 路由定义
└── core/            # 核心（依赖注入）
```
